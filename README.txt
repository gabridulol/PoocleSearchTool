========== EXECUTANDO O POOCLE SEARCH TOOL ==========
- Para executar o Poocle Search Tool instale os seguintes itens:
        * Python - https://www.python.org/
            * Biblioteca NLTK - https://www.nltk.org
        * Meson Build - https://mesonbuild.com/
        * GIMP Tool Kit - https://www.gtk.org/
========== UTILIZANDO O POOCLE SEARCH TOOL ==========
- Para utilizar o Poocle Search Tool siga as seguintes etapas:
        Etapa 1 : inserir o arquivo de entrada (Insert source file)
        Etapa 2 : construit o índice invertido dos documentos (Build inverted index)
- É necessário que você insira o arquivo de entrada primeiramente e 
depois construa o índice inverido para o funcionamente adequado do 
programa. Utilize as demais funções conforme necessários depois de 
concluir as etapas 1 e 2.
